# STARSHOT

Pixel-art 2.5D top-down shooter for PC. Runs in any browser — Windows, Linux, macOS.

## Key features

- **Works offline** — download and play from a flash drive, no internet needed
- **Lightweight** — small source footprint, fast to load
- **Cross-platform** — any PC with a browser works
- **Local multiplayer** — co-op and PvP over LAN (in dev)
- **Online play** — dedicated server version with rooms for remote players (in dev)
- **Flexible settings** — adjust enemy strength, give handicap to selected players, tweak to your liking (in dev)
- **In-game debug terminal** — full console for advanced tweaks and debug
- **No external dependencies** — pure vanilla JS, HTML5 Canvas, CSS

## How to play

- **Online:** https://jazz-crab.github.io/starshot/
- **Offline:** Open `index.html` in any browser

## Controls

- <kbd>W</kbd> <kbd>A</kbd> <kbd>S</kbd> <kbd>D</kbd> — Move
- <kbd>Mouse</kbd> — Aim
- <kbd>LMB</kbd> — Shoot
- <kbd>RMB</kbd> — Special
- <kbd>R</kbd> — Reload
- <kbd>Q</kbd> — Ult
- <kbd>Space</kbd> — Dash
- <kbd>Escape</kbd> — Pause
- <kbd>/</kbd> or <kbd>></kbd> or <kbd>~</kbd> — Terminal
