# STARSHOT

Pixel-art 2.5D top-down shooter for PC. Runs in any browser — Windows, Linux, macOS.

## Key features

- **Works offline** — download and play from a flash drive, no internet needed
- **Lightweight** — small source footprint, fast to load
- **Cross-platform** — any PC with a browser works
- **Local multiplayer** — co-op and PvP on the same machine
- **Online play** — dedicated server version with rooms for remote players
- **Flexible settings** — adjust enemy strength, give handicap to selected players, tweak to your liking
- **In-game debug terminal** — full console for advanced tweaks
- **No external dependencies** — pure vanilla JS, HTML5 Canvas, CSS

## How to play

- **Online:** https://jazz-crab.github.io/starshot/
- **Offline:** Open `index.html` in any modern browser.

## Controls

WASD — move | Mouse — aim | LMB — shoot | RMB — special | Q — ult | Space — dash
