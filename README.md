# STARSHOT

Pixel-art 2.5D top-down shooter for PC. Fully cross-platform — runs in any modern browser on Windows, Linux, and macOS.

## Key features

- **Works offline** — download and carry the game with you on a flash drive; no internet required.
- **Lightweight** — tiny source footprint leads to instant loading.
- **Cross-platform** — runs on any PC with a modern web browser.
- **Local multiplayer** — co-op and PvP over LAN (in development).
- **Online play** — dedicated server version with rooms and cross-device progress synchronization (in development).
- **Flexible settings** — adjust enemy strength, give a handicap to beginners, and tweak the balance to your liking (in development).
- **In-game terminal** — built-in console for instant configuration and debugging.
- **No external dependencies** — pure vanilla JS, HTML5 Canvas, and CSS.

## How to play

- **Online:** https://jazz-crab.github.io/starshot/
- **Offline:** Open `index.html` in any modern browser

## Controls

- <kbd>W</kbd> <kbd>A</kbd> <kbd>S</kbd> <kbd>D</kbd> — Move
- <kbd>Mouse</kbd> — Aim
- <kbd>LMB</kbd> — Shoot
- <kbd>RMB</kbd> — Special ability
- <kbd>R</kbd> — Reload
- <kbd>Q</kbd> — Ult
- <kbd>Space</kbd> — Dash
- <kbd>Escape</kbd> — Pause
- <kbd>/</kbd> or <kbd>></kbd> or <kbd>~</kbd> — Terminal
