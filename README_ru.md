# STARSHOT

Пиксельный 2.5D шутер для ПК. Полная кроссплатформенность — работает в любом современном браузере под Windows, Linux и macOS.

## Ключевые особенности

- **Работает офлайн** — скачайте и носите игру с собой на флешке, интернет не нужен.
- **Облегчённый** — минимальный размер исходного кода -> мгновенная загрузка.
- **Кроссплатформенность** — подойдет любой ПК с современным браузером.
- **Локальный мультиплеер** — кооператив и PvP по локальной сети (в разработке).
- **Игра по сети** — версия с выделенным сервером и комнатами для удаленных игроков, синхронизация прогресса между устройствами (в разработке).
- **Гибкие настройки** — регулируйте силу врагов, давайте фору новичкам, настраивайте баланс под себя (в разработке).
- **Внутриигровой терминал** — консоль для мгновенной настройки и отладки.
- **Никаких внешних зависимостей** — чистый (vanilla) JS, HTML5 Canvas, CSS.

## Как играть

- **Онлайн:** https://jazz-crab.github.io/starshot/
- **Офлайн:** Откройте файл `index.html` в любом современном браузере

- <kbd>R</kbd> — Перезарядка
- <kbd>Q</kbd> — Ульта
- <kbd>Space</kbd> — Рывок
- <kbd>Escape</kbd> — Пауза
